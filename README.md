Docker image with  Java
============

Personal Java docker image based on Debian 9 (Stretch)

#### Versions/tags

All tags upgraded to `debian:stretch`

#### Tags

Oracle Java 8 Server-JRE:
* `8`

Oracle Java 9 Server-JRE:
* `9`

#### Usage

Example:

```
$ docker run -it --rm globdev/java:8 -version
```
